# Simulando um Malware de Captura de Dados Simples em Python e Aprendendo a se Proteger

> **AVISO:** Este repositório contém scripts desenvolvidos exclusivamente para fins educacionais e de demonstração de vulnerabilidades em ambientes controlados. O uso destes códigos para qualquer atividade ilícita é estritamente proibido e antiético.

Este projeto faz parte da formação em Cibersegurança da **DIO (Digital Innovation One)** e demonstra o funcionamento técnico de ameaças comuns e as respectivas contramedidas de defesa.

## 📂 Estrutura do Projeto

* **/Simulacao_MALWARE**: Contém os scripts `ransoware_simulado.py` (criptografia) e `descriptografar_simulado.py` (restauração), além da pasta de arquivos de teste.
* **/Simulacao_KEYLOGGER**: Contém o script `keylogger_simulado.py` para monitoramento de eventos de teclado.

## ⚠️ Aviso!!!!
O conteúdo deste repositório é para uso **estritamente educacional**. O objetivo é compreender a lógica de funcionamento de malwares para fortalecer a segurança defensiva e a proteção de dados.

## 🛡️ Medidas de Prevenção
* **Ransomware:** Manter backups offline e utilizar soluções de EDR (Endpoint Detection and Response).
* **Keylogger:** Utilizar autenticação de dois fatores (2FA) e antivírus atualizados para detecção de hooks suspeitos.
